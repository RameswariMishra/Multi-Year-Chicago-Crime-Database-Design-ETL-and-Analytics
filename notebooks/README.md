# Notebooks

This folder contains Jupyter notebooks implementing database design, ETL processing, and analytics on multi-year Chicago crime data.

## Contents

- **Multi-Year Chicago Crime Database Design, ETL, and Analytics Part1.ipynb**  
  Covers database schema design, data ingestion, cleaning, and ETL preparation for multi-year crime records.

- **Multi-Year Chicago Crime Database Design, ETL, and Analytics Part2.ipynb**  
  Focuses on analytical querying, trend analysis, and visualization of crime and arrest patterns across years.
